# SFMTA Special Event Parking iCal

When there are events in Oracle Park or Chase Center street parking in Mission Bay can get very expensive. This repo creates a calendar for people who work or live in Mission Bay and want an easy heads-up before driving there. It converts [SFMTA's special event parking schedule](hhttps://www.sfmta.com/notices/current-special-event-parking-regulations-schedule) into an iCal feed that you can subscribe to in your calendar app.

## Subscribe

Use this URL in your calendar app:

https://raw.githubusercontent.com/sitic/sfmta-special-event-parking-ical/main/sfmta_special_event_parking.ics

or this Google Calendar URL:

https://calendar.google.com/calendar/u/0/embed?src=rdd6dhu5ctb6qmmg63fg6m82ecfutrct@import.calendar.google.com
